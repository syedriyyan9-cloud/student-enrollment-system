# Project: Student Enrollment System

The Student Enrollment System is a web application that allows students to sign 
up and log in to access their dashboard. Students can view available courses and
request enrollment. The admin has full control over the system and can approve 
or reject student enrollments. The admin can also create, update, view, and 
delete student records and course data. In short, all CRUD (Create, Read, 
Update, Delete) operations are managed by the admin.

## Features
- Login & Sign Up
- Student Dashboard
- Course Enrollment Request
- Admin Approval / Rejection of Enrollments
- Student Management (CRUD Operations)
- Course Management
- Search Bar for Students or Courses

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
